## 🤝 Get Involved

There are many ways to contribute to V-COMM:

1. **Code** - Submit pull requests for bug fixes or features
2. **Documentation** - Help improve our documentation
3. **Testing** - Report bugs and test new releases
4. **Translation** - Help translate V-COMM to your language
5. **Support** - Help other users in Discord and GitHub Discussions
6. **Content** - Write blog posts, create videos, or give talks
