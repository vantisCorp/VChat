## 🔄 Upgrade Guide

### Upgrading from 0.1.0 to 0.2.0

```bash
# Backup your data
make backup

# Update dependencies
make update

# Run migrations
make db:migrate

# Restart services
make restart

# Verify upgrade
make check
```

See [UPGRADE.md](docs/upgrade.md) for detailed instructions.
